# 最小自测清单（主流程）

每次改动后按顺序做一遍，预计 3-5 分钟。

## A. 首页可交互
1. 打开首页，确认无白屏、无明显错位。
2. 点击“灵性环境音”，按钮文案在“🎵 灵性环境音 / 🔇 静音”间切换。
3. 输入问题后点击“开始问卜”，应进入仪式流程，不应卡死。

## B. 抽牌与翻牌流程
1. 进入抽牌区后，顶部提示数量正确。
2. 抽完指定数量牌后，自动进入翻牌提示。
3. 依次翻牌后，进入解读区域。

## C. 解读流与结果动作
1. 解读文字应逐步出现（流式）。
2. 接口失败时应显示“宇宙连接中断”提示，而不是页面无响应。
3. 解读结束后，“保存羊皮卷轴 / 复制解读 / 重新占卜”按钮可点击。

## D. 历史与持久化
1. 完成一次解读后，右侧“占卜记录”出现新记录。
2. 刷新页面后，记录仍存在（localStorage 生效）。
3. 点击“清空记录”后，显示“暂无记录”。

## E. 今日神谕
1. 点击“今日神谕”进入日签视图。
2. 牌面和日期渲染正常。
3. 无接口时应展示兜底文案，不应报错卡死。
